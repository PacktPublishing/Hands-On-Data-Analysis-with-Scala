## Instructions and Navigations
All of the code is organized into folders. For example, Chapter02.

The code will look like the following:
```
scala> def factorial(n: Int): Long = if (n <= 1) 1 else n * factorial(n-1)
factorial: (n: Int)Int
scala> factorial(5)
res0: Long = 120
```
